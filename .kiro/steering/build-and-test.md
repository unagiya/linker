# ビルドとテストのガイド

このドキュメントは、プロジェクトのビルドとテストの実行方法を説明します。

## ビルド方法

プロジェクトのビルド手順:

```bash
# 依存関係のインストール
npm install

# ビルドの実行
npm run build
```

## テストの実行

### すべてのテストを実行
```bash
npm test
```

### 特定のテストファイルを実行
```bash
npm test -- <test-file-path>
```

### ウォッチモードでテストを実行
```bash
npm test -- --watch
```

## 開発サーバー

開発サーバーの起動:

```bash
npm run dev
```

## リンティング

コードの品質チェック:

```bash
npm run lint
```

## トラブルシューティング

### よくある問題

1. **依存関係のエラー**: `node_modules`を削除して再インストール
2. **ビルドエラー**: キャッシュをクリアして再ビルド
3. **テストの失敗**: テストの依存関係を確認
